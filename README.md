# PokemonClicker
---
## Ejecucion : 

Primero es necesario instalar las dependecias de node.

Puedes usar :

`npm install`
o
`npm i `

Una vez con todas las depencias listas , la app estaria lista.

Para ejecutar el el clicker:

En la raiz del proyecto ejecutamos : 

`npm run dev`

**¡ Et voilà !** , ya tienes tu clicker.
